HTML

HTML (HyperText Markup Language) je standardni jezik koji se koristi za izradu web stranica. To je osnovni jezik koji se koristi za strukturiranje sadržaja na webu. HTML koristi oznake ili tagove kako bi definirao različite elemente na web stranici, poput naslova, paragrafa, slika, veza itd.

HTML je skraćenica za "HyperText Markup Language". HyperText znači da web stranice mogu sadržavati veze ili hyperlinkove koji vode na druge stranice ili resurse, dok "Markup Language" znači da se koristi za označavanje ili definiranje strukture dokumenta koristeći oznake.

HTML je ključna tehnologija weba zajedno s CSS-om (Cascading Style Sheets) i JavaScriptom. CSS se koristi za stiliziranje i dizajniranje web stranica, dok se JavaScript koristi za interaktivnost i dinamičko ponašanje.
